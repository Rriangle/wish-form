GameCore 遊戲論壇平台｜詳細需求與系統架構

一、平台定位與開發技術

GameCore 是一個以「遊戲論壇」為核心，整合社群互動、點數經濟、虛擬寵物飼養、每日任務獎勵、小遊戲打怪、商城與玩家市集、排行分析、以及遊戲約玩配對等模組的互動型遊戲社群平台。所有功能模組皆透過點數機制相互串聯，提供會員長期參與誘因。

  • 後端：ASP.NET (MVC) + C#
  • 資料庫：SQL Server
  • 前端：HTML / CSS / JavaScript（支援 Vue / Tailwind / shadcn）
  • 登入機制：Email、自訂帳號 + OAuth（Google、Facebook、Discord）

二、使用者功能（前端）

1. 會員系統

  • 註冊 / 登入 / 登出 / 忘記密碼
  • 支援第三方登入：Google / Facebook / Discord
  • 第三方帳號綁定、切換與解除
  • 個人資料編輯、頭像、暱稱、點數紀錄

2. 每日簽到

  • 每日簽到可得固定點數
  • 連續簽到提供額外獎勵
  • 顯示簽到歷史與連續天數

3. 虛擬寵物系統（史萊姆）

  • 每位會員皆可飼養 1 隻史萊姆寵物
  • 寵物可餵食、洗澡、逗玩、生病治療
  • 寵物具有屬性與等級，互動會提升經驗值
  • 升級後玩家可獲得點數獎勵
  • 寵物有留言板供玩家或其他用戶互動

4. 小遊戲系統（冒險打怪）

  • 每日最多可冒險 3 次
  • 點擊「出發冒險」隨機生成怪物
  • 比對寵物等級 + 隨機因子 vs 怪物 → 決定勝負
  • 勝利者可獲得隨機點數
  • 戰鬥會記錄於冒險歷史紀錄中
  • ⚠️ 小遊戲與寵物關係：戰鬥時使用的是玩家的虛擬寵物（史萊姆）。戰鬥結果影響寵物經驗值、等級與成長，進而影響玩家點數收入。

5. 商城與玩家市集

  • 官方商城（新台幣支付＋點數折抵）：銷售遊戲周邊、點數卡
  • 玩家市集（新台幣支付＋點數折抵）：（1）可上架遊戲虛寶拍賣（平台抽手續費）（2）與玩家媒合交換虛寶
  • 點數折抵機制全站適用

6. 論壇模組

  • 多分區討論版：英雄聯盟、原神、綜合、心情版等
  • 發文、留言、編輯、刪除功能
  • 貼文可加上多個標籤
  • 投票功能（需扣點）

7. 排行榜模組

  • 綜合遊戲熱度排行（從網路爬蟲抓取 Steam / Twitch 數據）
  • 各遊戲分區熱門排行
  • 會員虛擬寵物（史萊姆）等級排行榜

8. 約玩系統

  • 玩家可發起約玩房（指定遊戲 / 時間 / 聯絡方式）
  • 開放配對與留言招募機制
  • 設有聊天室/即時留言功能

三、後端管理模組（管理者）

  • 會員管理：查詢、封鎖、黑名單、等級調整
  • 貼文管理：檢舉審核、刪除貼文、編輯分類
  • 商城 / 市集管理：商品上架、下架、市集抽成調整
  • 活動與公告管理：簽到設定、節慶點數加成
  • 排行榜資料來源維護與快取
  • 小遊戲管理：怪物資料、點數掉落設定

四、三層式架構設計（ASP.NET MVC）

  • UI層（前端）：Razor / HTML / Vue 組件設計頁面與互動
  • 邏輯層：各功能模組控制器（例如 SignInController, PetController）
  • 資料層：透過 EF Core 存取 SQL Server，如 PetRepository, UserRepository
